CVV2/CSC is required for eCommerce payments when the cardholder is not presented to sign the receipt. It is commonly used as a security tool for phone or online orders, in order to ensure the purchaser is in possession of a genuine card. For VISA, MasterCard and Diners this number (3 digits) is printed on the signature area at the back of the card. For AMEX the CSC-Card Security Code (4 digits) is located on the front of the card, above the credit card number. Most of the cases it will be on the right corner of the card.

The name of this code differs per card network. You may also know it as the Card Verification Code (CVC2), Card Verification Value(CVV2), Card Identification Number (CID) or the Card Security Code.
